Project 2: International Mortality Analysis 
Project Description:
A tool for observing trends in mortality rates in different countries and then comparing with other statistics such as healthcare/phamaceutical expenditure.

Technologies Used
Apache Spark - version 3.2
Apache Hive - version 2.3.9
Scala Build Tool - version 1.5.5

Features

Admin/User login system
Looping user interface
Spark queries comunicating with Hive database
Aggregate data was visualized via Zeppelin
